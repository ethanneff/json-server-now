# Json Server Now

GraphQL and REST APIs using JSON Server backend hosted on now.sh

## Local

```sh
yarn
yarn start
```

http://localhost:5000

## Remote

```sh
yarn global add now
now
```

https://json-server.ethanneff.now.sh
